The IMLS Open Science Curriculum project has reached a significant milestone: 14 new lessons are now in active development, each piloted with real learners and moving toward formal adoption.

The curriculum trains librarians as open science educators — a train-the-trainers model that multiplies institutional capacity without requiring every library to build expertise from scratch.

Key progress:

- Lesson topics cover FAIR data, software citation, reproducibility, and open access
- Each lesson includes instructor notes and formative assessment
- Several lessons are on a path toward Library Carpentry adoption

Read the full update on the project site. The materials are freely available under CC-BY.
